In de 2de themaweek in periode 3 heeft de klas de opdracht gekregen om zelf een spel/applicatie te maken. Ik heb ervoor gekozen om het spel: Snake, te maken. Een andere manier hoe je af kan zijn, is als je jezelf raakt. Het is niet de bedoeling om je eigen slangen lichaam aan te raken en in de knoop te zitten. Wil jij deze game spelen? Webcatalogus

In de 3de periode heeft de klas een project gekregen, genaamd: "Webcatalogus". Ik heb dit project gemaakt met dezelfde 2 vrienden waarmee ik ook het project "Buid A Game heb gemaakt". Het is de bedoeling dat je gaat werken met een tak van Javascript genaamd: "JSON" Je moet met deze JSON, data gaan zetten van dingen die jij zelf heb gekozen die jij erin wil zetten. Ook hebben wij geleerd om met NodeJS te werken. Met NodeJS kan je JavaScript buiten een browser uitvoeren. Hiermee kan je dus jouw JSON data laten zien aan derest van de wereld. In het project moeten wij een hoofdpagina hebben en daarin 3 subcategroieën waar de gebruiker van de pagina uit kan kiezen. Ook moest er een contactformulier gemaakt worden. Die hebben wij gemaakt met HTML. Wil je zien hoe onze project eruit ziet? Klik dan op de foto hierboven. Heel veel plezier!

Dashboard

In de 4de periode heeft de klas een project gekregen, genaamd: "Dashboard". Ik heb dit project gemaakt met dezelfde 2 vrienden waarmee ik ook het project "Buid A Game" en "Webcatalogus" heb gemaakt. Het is de bedoeling dat je gaat werken met ChartJS, MongoDB en Bootstrap. Met ChartJS kan je 8 verschillende soorten diagrammen mee maken, de: staaf, lijn, gebied, taart, bel, radar, polair en verstrooiing diagrammen MongoDB is een database waar je jouw data op kan slaan die jij in MongoDB heb gezet. Het is geschreven in de programmeertaal: C++. Bootstrap is een verzameling hulpmiddelen voor het maken van websites en webtoepassingen. Het bevat sjablonen gebaseerd op HTML en CSS voor typografie, formulieren, knoppen, navigatie en andere interfaceonderdelen. Het bevat ook JavaScript-extensies. Wij moeten data verzamelen en in MongoDB zetten. Daarna moeten wij met ChartJS onze diagrammen mee maken. Ik heb ervoor gekozen om een staafdiagram, taartdiagram, lijngrafiek en een staafdiagram met meerdere onderdelen te maken. Het thema van mijn diagrammen is: F1 in het 2021 seizoen. Ik vond het heel erg lastig om dit project te maken, maar ik heb er het beste van gemaakt. Wil je zien hoe ons "Dashboard" project eruit ziet? Klik dan op de foto hierboven. Heel veel plezier!
